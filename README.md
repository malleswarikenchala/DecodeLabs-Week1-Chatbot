# Rule-Based AI Chatbot

## Objective

To build a simple rule-based chatbot using Python that responds to predefined user inputs.

## Technologies Used

* Python

## Features

* Greets users
* Responds to basic questions
* Handles exit commands
* Provides default responses for unknown inputs

## How It Works

The chatbot uses conditional statements (`if`, `elif`, `else`) to match user input with predefined responses.

## Supported Commands

* hello
* hi
* how are you
* what is your name
* bye
* exit

## How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run:

python3 chatbot.py

## Sample Output

AI Chatbot Started!
Type 'exit' to stop the chatbot.

You: hello
Bot: Hello! How can I help you?

You: exit
Bot: Goodbye! Have a nice day.

## Conclusion

This project demonstrates the basics of Artificial Intelligence using a rule-based approach and helps understand chatbot logic using Python.
